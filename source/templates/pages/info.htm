What is SynchroVideo?

SynchroVideo is a webservice in the spirit of the late synchtube.com. The site is divided into "rooms", where multiple people can connect and watch videos together. One person acts as the "host", and can control video playback, while anyone else in the room is automatically synchronized to the host's position in the video.

How do I join a room?

On the home page you can search for rooms. The room search will find rooms based on their name and tags. The tags for a room can be set in the "[Room Settings]" page of each room by anyone with the room's host password. Searches will also look at and show the current video playing in the room, so you can find rooms by their playlists.

If you already know the name of the room you want to join, you can go to its URL directly. The format for room URLs is

http://synchrovideo.com?action=join&room=ROOM_NAME

You can also use

http://synchrovideo.com/room/ROOM_NAME

or

http://synchrovideo.com/r/ROOM_NAME

to go to the room's page

What video formats are supported?

Currently, SynchroVideo supports only YouTube videos/playlists, but support for other video-sharing sites is planned.

Is this project open-source?

Yes, you can view the latest source code here.

How do I create a room?

On the home page, under Tools, click Create Room. Enter the CAPTCHA text (to prevent spam) into the box below the generated CAPTCHA picture. If the CAPTCHA is too difficult to read, try generating a new one by hitting the "Reset" button or refreshing the page.

In the "Room Name" box, enter the full title that you want for your room. This name should be 5-16 alphanumeric characters long and may also contain spaces.

In the "Host Pass" box, enter the password you will use to gain "host" in your room.

Finally, hit the "Submit" button. If the settings you specified are valid, your room will be created and you will be automatically redirected to its URL. The name of the room in the URL may be slightly different from the full room name, so be sure to make a note of it.

How do I customize my room?

You can customize some aspects of your room, including a banner, the page title, and eventually more. You can also make your room private, which means clients will need a password to join it. Private rooms also do not appear in searches for rooms on the home page.

To edit your room's settings, look in the "Settings" section near the bottom of the page, for "[Room Settings]". Clicking the link will open the room configuration page (if you are not currently the host, you will need to enter the room's host password to gain access).

To make your room private, simply enter a password into the "Room Password" box (to make it public again, make this box blank). Do not use the same password for the room and host, unless you trust everyone who will enter the room!

To add a banner, paste a link to an image in the "Banner" box. The image should be 1000x300 pixels or else it will be stretched. To remove the banner, simply leave the "Banner" box blank.

After you've made the changes you want, click "Save Changes" to apply them to the room. The configuration window should then automatically close.

Custom room scripts

You can customize your room even further by using custom CSS and Javascript code. In the Room Settings page, point the Custom Script to a Javascript file and the Custom Style to a CSS file (you will need to host these elsewhere).

The "#sandbox" div is especially intended as a place to drop custom content on to the page, although there's almost no limit to how you can customize the page.

Be warned: harmful or annoying scripts will be removed.

How do I use a room?

Regular client

As a regular client (i.e. not the room host), most things are already automated for you. You can join the room's chat by entering a nickname in the IRC chat box below the video and clicking the connect button.

If the room's Playlist is in "unlocked" mode, then you may add videos to it via the "Add" box below the playlist. If it is locked however, only the host can add new videos.

You can adjust the size of the video in the Settings section near the bottom of the page. This can help accomodate smaller/larger computer screens. You can also change the position of the video, chat, playlist, and controls by choosing a different layout from the Layout drop-down list.

Host

To become the room's host, simply click the "Become Host" button in the "Settings" section near the bottom of the page.

As the host, you will notice a larger number of controls below the playlist:

  • Clear  removes all videos from the playlist.
  • Lock  secures the playlist so only the host can add videos.
  • Unlock  opens the playlist so anyone can add videos.
  • Shuffle  mixes the order of the playlist randomly.
  • Random  plays a random video from the playlist.
  • Sort  arranges the playlist alphabetically by video title.
  • TV Mode  lets the server act as host, so the room will continue to play through the playlist without any real host.

The "List" section allows you to save or load created playlists for the room, or to import an existing YouTube playlist.

To save a playlist, enter a name for it in the List box, then click the Save button. NOTE: This will overwrite an existing playlist by the same name.

To load a playlist, select one from the drop-down menu below the List box or enter its name manually, then click the Load button.

To delete a playlist, select one from the drop-down menu below the List box or enter its name manually, then click the Delete button.

To import a YouTube playlist, paste the playlist URL into the List box,then click the Import button.

Keep in mind that loading/importing videos from a playlist will append the videos to the current playlist. Use the Clear button to start fresh before loading/importing if you wish.

Problems with synchronization?

If your video seems to stutter and jump around, you may be having trouble keeping up with the host. Slower connections require more time to synchronize, and this time can be adjusted in the "Settings" section near the bottom of the page. Most of these only appear for normal clients.

Firstly, if the video appears to be looping the same few seconds over and over again, then the room is most likely without a host, and there's nothing you can do as a normal client. Hosts can leave their room in TVMode to avoid this, but that's up to them.

The "Synchronize" checkbox controls whether or not your client synchronizes with the host at all. Unchecking this will solve any sync issues, but you may be widely out of sync with others in the room. You should only uncheck this box if nothing else helps.

"Time Buffer" controls a "grace period" between your video time and the host's video time. For example, a buffer of "1 sec" means your video time will only be forced to change if you are greater than 1 second behind or ahead of the host. If the video seems to jump around, you should try and increase this buffer. Ideally though, it should be as low as possible while providing smooth playback to ensure the closest synchronization between clients.

"Sync Interval" controls how often your client attempts to synchronize with the host. Unless maxing the Time Buffer doesn't help, you should leave this at the default value.
